Hey, ik ben jouw_naam.

Derdejaarsstudent Toegepaste Informatica aan Howest Brugge. Mijn interesse ligt op de grens tussen hardware en software — van blinkende LEDs tot complete IoT-systemen die met cloud APIs praten.

Ik hou van het moment waarop een microcontroller voor het eerst je code uitvoert, van een mooi gerouteerde PCB, en van firmware die elegant blijft binnen 32KB flash. Op de software-kant: webapps, dashboards en alles wat zo'n embedded device tot leven brengt.

Op zoek naar een stage waar embedded én software samenkomen. Klaar voor 2026.
